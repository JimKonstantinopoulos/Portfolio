# How to use
1. **Clone the repository**
    ```
    git clone https://github.com/JimKonstantinopoulos/Portfolio.git
    ```
2. **Navigate to the project directory**
    ```
    cd Portfolio
    ```
3. **Install dependencies**
    ```
    npm i
    ```
4. **Start the development server**
    ```
    npm run dev
    ```
